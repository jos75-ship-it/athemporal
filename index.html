Preguntas frecuentes

Información clave antes de cotizar.

Respuestas rápidas para los usuarios que llegan con intención comercial, pero aún necesitan seguridad antes de escribir.

¿La colección es para venta minorista o mayorista?
La propuesta está pensada principalmente para mayoristas, tiendas, proyectos de interiorismo y distribuidores.
¿Los valores incluyen IVA?
Las tablas de referencia indican valores + IVA. Las condiciones finales deben confirmarse directamente con el equipo comercial.
¿Qué modelos tienen precio publicado?
El catálogo incluye precios de referencia para M101 y M102. El M103 se presenta con medidas y especificaciones, pero requiere cotización.
¿Se puede pedir asesoría para escoger modelo?
Sí. El sitio está preparado para recibir solicitudes con modelo, cantidad y tipo de proyecto, facilitando una recomendación más precisa.
